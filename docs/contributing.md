# Contributing to City Smart Heaven

Welcome! This is a learning group project — all skill levels are encouraged to contribute. 🎉

## Getting Started
1. Clone the repository
2. Initialise submodules:
   ```bash
   git submodule update --init --recursive
   ```
3. Run the docs locally with Docker:
   ```bash
   docker compose up
   ```
4. Open [http://localhost:8000](http://localhost:8000) in your browser

## Ways to Contribute
- Documentation improvements
- New feature proposals (copy `docs/features/_template.md`)
- 3D printing / electronics / embedded code
- Reviewing merge requests

## Workflow
- Create a feature branch from `main`
- Follow our [software standard](build-standards/software-standard.md)
- Submit a Merge Request and request a review
- Ensure your work meets the [Definition of Done](process/definition-of-done.md)
