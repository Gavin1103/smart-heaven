# Districts (aka tiles)

We sometimes call each tile a “district” because it sounds nicer — but Smart Heaven should feel like one cohesive city. Keep visuals and wiring standards consistent so any tile can join the main line seamlessly.

Initial districts / tiles (T1–T5):
- T1 — Main street (traffic lights showcase)
- T2 — Parking bays
- T3 — Buildings + green space
- T4 — Bridge section
- T5 — Railway crossing

Naming note
- Use T1–T5 in docs and code; you can give each a friendly nickname on the page.
