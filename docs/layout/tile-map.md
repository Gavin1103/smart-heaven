# Tile map

Initial layout: 5 tiles in a single straight road (line) so we can expand sideways later.

Tile size: 30×30×0.36 cm plywood. Roads are laser‑engraved.

Planned features to place along the line:
- A normal road crossing for the traffic lights showcase
- A T‑split to branch into a side street (future expansion)
- A small bridge section
- A railway crossing with barrier

Notes
- Start simple: straight roads for all 5 tiles now; swap tiles later as features mature
- Keep cable routing consistent so tiles remain swappable
- Leave space near edges for future expansion pieces (curves, T, crossings)
