# Build standards

Clear, shared rules so every module fits the city, is safe to demo, and is easy to maintain. Use these when you design, print/laser, wire, and write code.

!!! tip "How to use this section"
- Start with the Tile standard so your footprint and mounting match the city
- Check Electronics & wiring for voltages, labels, and connectors
- Follow 3D print & laser guidance for tolerances and materials
- Apply the Software standard for structure, config, and messaging

## Quick checklist before a demo
- [ ] Footprint fits the tile and leaves ≥ 10% green space
- [ ] Wiring labeled; safe routing; strain relief where needed
- [ ] Reset procedure written on the feature page
- [ ] Config values documented (timings, thresholds, angles)
